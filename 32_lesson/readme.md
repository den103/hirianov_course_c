## Адреса и указатели

### Операция взятия адреса &.

Создадим переменную **i** со значением 10.

```c
int i = 10;
```

- **&i** - Операция взятия адреса переменной i. Она возвращает **адрес места в памяти**, где хранится значение i.

Тип этой операции записывается как **int***.

Адрес это числовое значение, которое обозначает положение в памяти. Но в языке С адрес не считается обычным числом. Он имеет специальный тип - **тип указателя**. Этот тип должен быть совместим с типом объекта на который он указывает. 

Создадим переменную **pi** типа **int*** и положим туда адрес **i**.

```c
int* pi = &i;  // можно и так int *pi = &i;
```

Переменная типа int* (pi) называется **указателем на int**.

При выборе стиля записи указателей необходимо учесть следующий ньюанс:
```c
int* ptr1, ptr2;   // ptr1 - указатель, ptr2 - ОБЫЧНЫЙ int!
int *ptr1, *ptr2;  // оба - указатели
```

Если к имени типа добавить *, мы получаем тип указателя на этот тип.
Таким образом, мы можем получить и адрес указателя pi — это будет указатель на указатель на int:

```c
int** ppi = &pi;
```

### Операция разыменования *.

Операция **разыменования** это фактически **получение значения** по адресу.

#### Базовый пример:

```c
#include <stdio.h>

int main() {
    int x = 42;        // обычная переменная
    int *ptr = &x;     // указатель хранит адрес x
    
    printf("Адрес x: %p\n", &x);      // 0x7fff... (адрес)
    printf("Значение ptr: %p\n", ptr); // тот же адрес
    
    // РАЗЫМЕНОВАНИЕ:
    printf("Значение по адресу ptr: %d\n", *ptr); // 42
    
    return 0;
}
```

#### Изменение значения через разыменование:

```c
int main() {
    int x = 10;
    int *ptr = &x;
    
    printf("До: x = %d\n", x);    // 10
    
    *ptr = 20;  // ИЗМЕНЕНИЕ значения через разыменование
    
    printf("После: x = %d\n", x); // 20
    
    return 0;
}
```

#### Разыменование и арифметика указателей:

```c
int arr[3] = {10, 20, 30};
int *ptr = arr;  // ptr указывает на arr[0]

printf("*ptr = %d\n", *ptr);       // 10 (arr[0])
printf("*(ptr + 1) = %d\n", *(ptr + 1)); // 20 (arr[1])
printf("*(ptr + 2) = %d\n", *(ptr + 2)); // 30 (arr[2])
```
