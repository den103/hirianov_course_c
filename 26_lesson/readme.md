## Функции в языке Си

**Функция** в языке Си - это блок кода, который выполняет определённую задачу и может быть вызван из других частей программы.
#### Функции можно 
- Объявить (declare)
- Определить (define)
- Вызвать (call)
- Взять адрес

**Объявление функции** сообщает компилятору о существовании функции: её имя, тип возвращаемого значения и параметры. Оно не содержит тела функции.


**Определение функции** это тоже самое что и **описание функции** выглядит следующим образом:

```
тип имя (список формальных параметров)
    {
        тело функции
    }
```
**Тип возвращаемого значения** должен быть известен на момент описания функции и это не может быть массив. Это может быть любой из простых типов или пользовательский тип данных.

Чтобы **функцию вызвать** необходимо, обратиться по её имени, после этого имени написать скобки и туда поставить **фактические значения параметров** (есть ещё и формальные параметры).

Функции могут вызываться синхронно и асинхронно. **В языке Си всегда происходит только синхронный вызов функций.** При этом, функция которая вызывает следующую функцию останавливается и ждём, сохранив данные и адрес с которого была вызвана функция в **стек**. Затем функция которую вызвали, возвращается по этому адресу который записан в стеке. Т.о. функции имеют важнейшую функциональность, которая позволяет им **возвращаться туда, откуда её вызвали**.
